Installation Instructions: (Read carefully)

Note: If you encounter any issues with a previous installation / crack, please
uninstall Master collection and delete those folders :

C:\Program Files (x86)\Common Files\Adobe\SLCache
C:\ProgramData\Adobe\SLStore


1.) Make sure your computer has NO INTERNET ACCESS!
    Disable your Network card and/or disconnect your Network cable.
    Disconnect from your wireless connection (if you have one).
    And make sure you dont have any of those entries in your hosts file
    127.0.0.1 lmlicenses.wip4.adobe.com
    127.0.0.1 lm.licenses.adobe.com
    Lets start clean.

2.) Download the Adobe Master Collection CS6 Trial from Adobe's website and save it to a folder on your PC.

3.) Launch the Adobe Master Collection CS6 installer.

4.) When asked for a serial number, launch the keygen (DO NOT CLOSE OR EXIT THE KEYGEN UNTIL YOU HAVE COMPLETED STEP)
    and press the 'GENERATE' button. (DO NOT CLOSE OR EXIT THE KEYGEN YET)

5.) Copy that serial number and then either:
	a.) Paste (ctl + v for Windows, Command-V for MAC) into the first textbox for the serial in the Adobe installer
            (the whole serial should paste across the textbox fields)

	OR, if you couldn't paste it all in at once..

	b.) Carefully type the serial into the Adobe installer.
   	c.) When the error "Please connect to the internet and retry" shows click connect later.

6.) Continue with the installation.

7.) Once Adobe Master Collection CS6 is installed, open any program. You will be prompted to Activate.

8.) Click on the "I'm having Internet Problems" OR "Activate Offline" button/link, then,.. if needed,.. click on the 'Generate Request Code' button.

9.) A Request Code will be displayed.
    Copy that code, including the "=" at the end of the code,

10.) Go back to the keygen and delete or select all the text in the "Request" field,... then paste your Request Code into that field.

11.) Click the 'GENERATE' button again, and an Activation code will be generated.

12.) Copy the Activation Code, and then paste it into the Adobe Activator window, and then validate.

13.) Widows Users: Execute  disable_activation.cmd (double click on it) 
     (in Vista or Win7, run it as admin if you have uac enabled)

     Or do it manually 
     Add the text below to the bottom of your hosts file. 
     (The host file is located C:\windows\system32\drivers\etc\hosts)


     MAC Users: Run Osx/disable_activation_osx as root
     if you dont know how to do that, open a terminal window
     then issue sudo -s, a root shell will open
     then simply do "sh disable_activation_osx"  w/o the ""
     or ./disable_activation_osx( make it executable chmod 755 disable_activation_osx 
     you do that from the folder where disable_activation_osx is. (just copy it anywhere on your hdd) 

     Or do it manually
     Add the text below to the bottom of your hosts file. (/private/etc/hosts /et/hosts)
